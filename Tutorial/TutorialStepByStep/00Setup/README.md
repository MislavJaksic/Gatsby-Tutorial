### Using the Gatsby CLI

```
$: npm install -g gatsby-cli

$: gatsby telemetry --disable
$: gatsby --help
```

### Create a Gatsby site

Starter is a partially built site.  

```
$: gatsby new Gatsby-Starter-Project https://github.com/gatsbyjs/gatsby-starter-hello-world

$: cd Gatsby-Starter-Project

$: gatsby develop
```

#### View your site locally

```
http://localhost:8000/
```
